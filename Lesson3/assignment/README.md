- 第三题（加分题）：自学C3 Linearization, 求以下 contract Z 的继承线
- contract O
```
L(O) := [O]
```
- contract A is O
```
L(A) := [A] + merge(L(O), [O])
= [A] + merge([O], [O])
= [A] + [O]
= [A, O]
```
- contract B is O
```
L(B) := [B] + merge(L(O), [O])
= [B] + merge([O], [O])
= [B] + [O]
= [B, O]
```
- contract C is O
```
L(C) := [C] + merge(L(O), [O])
= [C] + merge([O], [O])
= [C] + [O]
= [C, O]
```
- contract K1 is A, B
```
L[K1] := [K1] + merge(L[A], L[B], [B, A])
= [K1] + merge([A, O], [B, O], [B, A])
= [K1, B] + merge([A, O], [B, O], [A])
= [K1, B, A] + merge([O], [O])
= [K1, B, A, O]
```

- contract K2 is A, C
```
L[K2] := [K2] + merge(L[A], L[C], [C, A])
= [K2] + merge([A, O], [C, O], [C, A])
= [K2, C] + merge([A, O], [O], [A])
= [K2, C, A] + merge([O], [O])
= [K2, C, A, O]
```

- contract Z is K1, K2
```
L[Z] := [Z] + merge(L[K1], L[K2], [K2, K1])
= [Z] + merge([K1, B, A, O], [K2, C, A, O], [K2, K1])
= [Z, K2] + merge([K1, B, A, O], [C, A, O], [K1])
= [Z, K2, K1] + merge([B, A, O], [C, A], [O])
= [Z, K2, K1, B] + merge([A, O], [C, A], [O])
= [Z, K2, K1, B, C] + merge([A, O], [A], [O])
= [Z, K2, K1, B, C, A] + merge([O], [O], [O])
= [Z, K2, K1, B, C, A, O]
```
